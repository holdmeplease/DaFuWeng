# DaFuWeng

大富翁——清华之旅

灵感来自于传统的大富翁游戏，在地图和人物中加入了清华大学元素

详细的游戏介绍与操作指南见“用户文档_大富翁清华之旅.pdf”


开发环境：

python3.6

pygame1.9.4

pycharm2018.3.4(windows x64)


游戏运行方法：

按照仓库中的文件结构使用pycharm运行dfw.py


操作指南：按鼠标左键开始游戏、按键盘“↑”键进行掷骰子与买地操作
